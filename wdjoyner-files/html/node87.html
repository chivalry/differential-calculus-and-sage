sage: t = var("t")

sage: t0 = pi/6
sage: y_x = diff(y,t)/diff(x,t)
sage: y_x
-2*sin(2*t)/cos(t)
sage: y_x(t0)
-2
sage: m = y_x(t0); x0 = x(t0); y0 = y(t0)
sage: X,Y = var("X,Y")
sage: Y - y0 == m*(X - x0)
Y - 1/2 == -2*(X - 1/2)
