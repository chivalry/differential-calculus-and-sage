General rule for differentiation

From the definition of a derivative it is seen that the process of differentiating a function consists in taking the following distinct steps:

General rule for differentiating4.5:

  • FIRST STEP. In the function replace by , giving a new value of the function, .

  • SECOND STEP. Subtract the given value of the function from the new value in order to find (the increment of the function).

  • THIRD STEP. Divide the remainder (the increment of the function) by (the increment of the independent variable).

  • FOURTH STEP. Find the limit of this quotient, when (the increment of the independent variable) varies and approaches the limit zero. This is the derivative required.
