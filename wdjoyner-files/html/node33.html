Introduction

We shall now proceed to investigate the manner in which a function changes in value as the independent variable changes. The fundamental problem of the Differential Calculus is to establish a measure of this change in the function with mathematical precision. It was while investigating problems of this sort, dealing with continuously varying quantities, that Newton4.1was led to the discovery of the fundamental principles of the Calculus, the most scientific and powerful tool of the modern mathematician.
